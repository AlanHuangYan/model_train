安装指南
重要提示：请在命令行中按以下步骤操作
第一步：检查当前 PyTorch 版本

在激活的虚拟环境中运行：

python -c "import torch; print(torch.__version__)"

如果版本号带有 +cpu，说明是 CPU 版
如果版本很老（如 1.x 或 2.0），可能不支持当前环境

第二步：卸载旧版本

为确保干净的环境，先卸载当前的 torch：

pip uninstall torch torchvision torchaudio
第三步：安装支持 CUDA 的 PyTorch

由于你的驱动是 CUDA 13.2，可以安装支持 CUDA 12.x 的 PyTorch。
推荐使用 CUDA 12.1 版本（最稳定）： 

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

也可以安装 CUDA 12.4 版本（更新）：
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124

第四步：验证安装

安装完成后，再次运行检查命令：

python -c "import torch; print('CUDA:', torch.cuda.is_available()); print('Version:', torch.version.cuda)"

如果显示 CUDA: True，则问题解决

RTX 5060 显卡特别说明

RTX 5060 是最新一代显卡，需要 CUDA 12.4+ 才能支持 sm120 架构。
如果 CUDA 12.1 不可用，请使用： 

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
